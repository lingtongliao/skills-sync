# Changelog

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-03-11

### Added
- Initial plugin scaffold for `skill-sync`.
- New skill `skills-update-manager` with check/diff/update workflow.
- Safe updater script with dry-run, backup, hash verification, and restore path.
- Manifest generator script to produce `catalog/skills-manifest.json`.
- Unit tests for version parsing and status detection.
